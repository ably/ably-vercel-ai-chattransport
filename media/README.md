# Media assets

Images for the README and the Vercel template gallery. This folder is not part
of the Next.js build (nothing here is imported or served), so it does not affect
the deployed app.

## Files

| File | Used by | Spec |
| ---- | ------- | ---- |
| `hero.png` | The repository [README](../README.md), embedded under the title. Renders on GitHub only. | Wide banner, ~1200–1280px wide. PNG or WebP, optimized. |
| `thumbnail.png` | The vercel.com/templates gallery card. Uploaded to Vercel, not read from this repo | 1.25:1 ratio, ~1200×960px. PNG. |
| `demo-screenshot.png` | Screenshot used in the repository [README](../README.md) for GitHub only. | |
| `open-in-new-tab-highlight.png` | Screenshot used in the repository [README](../README.md) for GitHub only. | |
